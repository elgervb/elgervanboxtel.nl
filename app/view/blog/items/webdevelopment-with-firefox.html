Webdevelopement with Firefox

When developing for the web, Firefox is still my favorite browser.

The standard tooling from Firefox is pretty impressive. It has support standard things like the console, HTML inspector, a debugger, profiler and network monitor. What Firefox offers more is:

  1. a style editor which allows you to save the changed css
  2. tools to inspect canvas contexts
  3. storage to show you your cookies, indexedDB, local and session storage
  4. web audio contextvisualizer and audio node inspector for the Web Audio API
  5. Highlighter for newly painted areas, to visualize what's being updated
  6. 3D view of your page. This is by far my favorite. Rotate and tilt the page until you've found the stacked layers you were looking for.
  7. a javascript scrathpad
  8. responsive design mode with simulated touch events
  9. a color picker
  10. a utility to take a fullpage screenshot

All can be activeted when pressing CTRL-SHIFT+Q and going to the settings in the right upper corner of the toolbox.

A comprehensive guide is avaliable on Firefox Developer Tools
