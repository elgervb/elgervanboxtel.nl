Firefox extensions

I've made a list of other Firefox extensions I use regularly:

  1. Adblock
    Adblock Plus blocks all annoying ads, and supports websites by not blocking unobtrusive ads by default
  2. All in one gestures
    This extension allows you to execute common commands using mouse gestures, rocker navigation, scroll wheel navigation and page scrolling.
  3. firebug
    irebug integrates with Firefox to put a wealth of development tools at your fingertips while you browse. You can edit, debug, and monitor CSS, HTML, and JavaScript live in any web page
  4. Codeburner
    CodeBurner is a Firefox add-on that integrates with Firebug, to extend it with reference material for HTML and CSS.
  5. Colorzilla
    Advanced Eyedropper, Color Picker, Gradient Generator and other colorful goodies
  6. Firebug autocomplete
    Firebug command line autocomplete
  7. Html validator
    HTML Validator is a Mozilla extension that adds HTML validation inside Firefox and Mozilla. The number of errors of a HTML page is seen on the form of an icon
  8. JSON viewer
    View JSON documents in the browser
  9. Measure it
    Draw a ruler across any webpage to check the width, height, or alignment of page elements in pixels.
  10. Mobile View
    MobileView shows a panel with the current Tab content in phone-like panel
  11. Screen grab
    Screengrab! saves webpages as images
  12. Scriptish
    The greatest user script engine on the Internet (a fork of Greasemonkey).
  13. SQlite manager
    Manage any SQLite database on your computer.
  14. Tamper data
    Use tamperdata to view and modify HTTP/HTTPS headers and post parameters
  15. Wappalizer
    Wappalyzer is a browser extension that identifies software on websites
  16. Web Developer
    The Web Developer extension adds various web developer tools to the browser
  17. UnMHT
    UnMHT allows you to view MHT (MHTML) web archive format files, and save complete web pages, including text and graphics, into a single MHT file in Firefox/SeaMonkey
  18. Firebug addons
    Add-ons that extend Firebug
